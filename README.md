# Indian Census Data Analysis using Python

## Overview
This project analyzes Indian census demographic data using Python.

## Objectives
- Data Cleaning
- Exploratory Data Analysis
- Correlation Analysis
- Linear Regression Modeling

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Dataset
Contains:
- Villages
- Towns
- Households
- Population
- Area

## Visualizations
- Correlation Heatmap
- Histograms
- Scatterplots
- Residual Analysis

## Machine Learning Model
Linear Regression

## Evaluation Metrics
- Mean Squared Error (MSE)
- R² Score

## Results
Model shows moderate predictive performance and identifies non-linearity in census data.
